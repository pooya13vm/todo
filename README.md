# todo-app
## General info
It as a training project to work on MVC design pattern

## Technologies
### Project is created with:
- [x] node.js
- [x] express.js
- [x] ejs (template engine)
- [x] Bootstrap 5
- [x] font-awesome
## Setup
To run this project, install it locally using npm:
- $ npm install
- $ npm start
